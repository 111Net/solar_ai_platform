Global AI-powered solar planning SaaS

## Features
- FastAPI backend API improvements
- React frontend dashboard
- AI engine for solar optimization
- Global electricity tariffs and ROI calculations
- Installer quotation system
- Dockerized deployment for cloud

## Architecture
- backend/ - FastAPI API, services, migrations
- frontend/ - React dashboard
- ai_engine/ - Solar optimization algorithms
- docs/ - Documentation
- docker/ - Docker deployment files

## Setup
1. Clone repo
2. Create Python virtual environment
3. Install dependencies: pip install -r backend/requirements.txt
4. Run backend: uvicorn backend.main:app --reload
5. Run frontend: npm start (inside frontend folder)
6. Dockerized deployment via docker-compose
